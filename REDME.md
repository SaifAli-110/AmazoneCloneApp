# The Amazon Clone project is a static front-end application that mimics the look and feel of   the world’s largest e-commerce platform. Without JavaScript or a backend, the focus is entirely on layout structure, navigation bars, and responsive product grids.

# Core Features to Build
# The Navigation Bar (Header):

# The "Amazon" logo placeholder.

# A search bar with a category dropdown and search icon.

# Accounts & Lists, Orders, and Cart icons.

# The Sub-Header:

# A secondary menu for "All," "Customer Service," "Registry," "Gift Cards," and "Sell."

# Hero Section:
# A large, sliding background image (or a static high-quality banner) that sets the theme for the site.

# Product Grid Layout:
# Multi-column cards containing product titles, images, prices, and "Shop Now" links.

# Footer:
# A "Back to top" button and multi-column links for company information and services.

# 🛠️ Technical Implementation
# The HTML Structure
# Your HTML acts as the "skeleton." You will use semantic tags to keep the code organized:

# <header> for the navigation area.
# <nav> for the links.
# <main> for the hero image and product cards.
# <footer> for the bottom links.

# The CSS Styling
# This is where the magic happens. To get that "Amazon look," you’ll use:

# Flexbox & CSS Grid: Essential for aligning the search bar and creating the responsive product boxes.

# Positioning: Using position: sticky to keep the header at the top while scrolling.

# Box Shadow & Hover Effects: To make product cards pop when a user mouses over them.

# Colors: Matching Amazon's specific palette, such as:

# Dark Blue/Black (#131921): Main Header.

# Light Blue (#232f3e): Sub-header.

# Yellow/Orange (#febd69): Search button and accents.

# 🎯 Learning Objectives
# By completing this project, you will master:

# Responsive Design: Ensuring the site looks good on both wide monitors and smaller laptops.

# Component Architecture: Learning how to treat sections (like a product card) as repeatable blocks.

# Icon Integration: Using FontAwesome or Google Icons for the cart and search features.
